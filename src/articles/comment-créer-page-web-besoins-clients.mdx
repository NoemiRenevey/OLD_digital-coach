## Un site web n'est pas un livre
Votre site web est une réponse (parmi d'autres réponses possibles) à une question que se pose un internaute - potentiellement un futur client. 
Personne ne lira jamais un site web de long en large : il faut que chaque personne, avec son besoin spécifique, tombe directement sur la bonne page qui réponde à la question qu'elle a tapée dans son moteur de recherche.
Cette page existe-t-elle chez vous ? 

## 1 page = 1 réponse = 1 action


Assurez-vous que chaque page de votre site réponde à une question précise, et une seule uniquement.
