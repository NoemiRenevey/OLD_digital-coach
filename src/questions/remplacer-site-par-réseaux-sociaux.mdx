# Peut-on remplacer son site web par des réseaux sociaux ? 
Vous avez peut-être entendu dire qu'avec les réseaux sociaux, il n'y a plus besoin de site web, ou pire, qu'avoir un site est has-been. Détrompez-vous : Le site web n'a jamais autant été au coeur de la machine digitale qu'aujourd'hui. 

## 5 différences clés entre réseaux sociaux et site web
1. Le site web vous appartient, vous êtes propriétaires de son code et du contenu qui y est posté. Sur le réseaux, non. 
2. Un site web est un outil de vente. Les réseaux sont des outils de communication dont la force est de créer du traffic sur votre site. 
3. Sur les réseaux, vous pouvez poster du contenu à l'infini, sur votre site, non. C'est pourquoi les réseaux sont un excellent outil de communication, au service de votre site, qui lui ne comprend que les biens et services essentiels à votre business.
4. Les contacts (communauté) que vous avez sur les réseaux ne vous appartient pas.  Sur votre site, les contacts emails que vous récoltez rentrent dans une base de donnée qui vous appartient
5. Sur votre site, vous êtes maître de votre image de marque, sur les plateformes, non. 

Pensez-y comme ceci : votre site est un restaurant, vos réseaux sociaux, les rabatteurs qui vous proposent le menu dans la rue. Les deux sont importants, mais il est quand même essentiel de conserver son restaurant... non ?
